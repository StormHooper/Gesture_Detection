Testing the Hand Detector

  1. Run "test.py" to test that the model works
  2. Hold different fingers and amounts of fingers
  3. Check if the console recorded:
    • Correct fingers held up
    • Correct number of fingers
    • An angle that is getting announced every second
  4. Move on to Data Collecting

Data Collecting

  1. Skip to Training the Data IF you are using the data already provided without additional
  2. Look up a picture of the sign language that you are using
  3. Run the "collect_asl_data.py"
  4. Hit the letter key that follows with the letter that data is being recorded for
  5. Hit the space bar to record about 50 samples for
    • EACH letter
    • EACH hand
  6. Move on to Training the Data

Training the Data

  1. Check the file type of the data, if it is not ".json", go back to the Data Collecting
  2. Make sure the data is named "asl_data.json" or the program will error
  3. Run "train_asl_model.py"
  4. If the graph shown has a lot of "1.00", then DELETE "mac_data.json" and redo Data Collecting
  5. Check that 2 new files were created:
    • "asl_knn_model.jotlib"
    • "label_encoder.jotlib"
  6. Move on to Testing the Model

Testing the Model

  1. Run "main.py"
  2. Test the different hand gestures
  3. Check that the console can correctly identify the gestures
  4. Activate the ASL detection mode
  5. Check that the console can correctly identify the ASL letters
  6. Feel free to suggest other gestures
